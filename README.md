Este projeto consiste no desenvolvimento de uma calculadora funcional em HTML, CSS e JavaScript, criada com foco em simplicidade, responsividade e uma experiência de usuário agradável.

A interface apresenta um visual moderno, com fundo em degradê e um painel central estilizado que destaca a calculadora. Os botões foram projetados para oferecer boa legibilidade, contraste e interação, incluindo um efeito hover para melhorar a usabilidade.

No lado funcional, a calculadora permite:

- Inserção de números e operadores matemáticos

- Limpeza completa da tela (clean)

- Remoção do último caractere (backspace)

- Cálculo automático da expressão inserida

- Operações básicas como adição, subtração, multiplicação e divisão

Este projeto evidencia habilidades como:

- Estruturação de páginas com HTML

- Estilização e criação de layout responsivo com CSS

- Manipulação de eventos e lógica matemática em JavaScript

- Organização e clareza no código

- Criação de experiências visuais simples e eficientes
